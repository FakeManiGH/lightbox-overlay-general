Overview

LightBox Overlay (LBO) is an “App" for creating image viewing layer on any webpage that supports JavaScript. LBO is designed to work with image galleries and single images. This page provides all you need to know about LBO.

→ LightBox Overlay is RESPONSIVE and works on all display sizes (Desktop, Tablet, Mobile).

Setup

  • Download latest release of LBO in .zip -file from GitHub. GitHub releases here.
  • From the .zip -file, export lbo.css and lbo.js to your website project folder. You can also create subfolder.
  • When you have the files in right place, you can link them to your HTML document.
    • Link the CSS file in <head> section of your HTML document. Example:
      <link rel="stylesheet" href="location/of/lbo.css">
    • JavaScript file can also be placed in <head> section of HTML document with defer -attribute. Example:
      <script src="location/of/lbo.js" defer></script>
  • When you have both files successfully linked on your document. Move to Activation.

Activation

  • To activate Lightbox Overlay script, you need some images on your HTML document. Add image(s).
  • Lightbox Overlay for a SINGLE IMAGE is activated by adding data-lbo attribute to an image-element.

    <img src="location/image.jpg" data-lbo alt="description">
  • To create a IMAGE GALLERY, you need to give same data-lbo="gallery name" to multiple images in your HTML document.

    <img src="location/image1.jpg" data-lbo="gallery1" alt="description">

    <img src="location/image2.jpg" data-lbo="gallery1" alt="description">

    <img src="location/image3.jpg" data-lbo="gallery1" alt="description">
  • To set title for your image inside LBO, add data-title="title here" attribute to image element.

    <img src="location/image.jpg" data-lbo alt="description" data-title="title Here">
  • Enjoy your Lightbox Overlay Gallery(s).

Download Function

To setup download function, simply add data-download="yes" attribute to an image-element.

<img src="location/image.jpg" data-lbo alt="description" data-download="yes">

User Manual

  • You can browse image galleries inside LBO by using Next and Previous buttons inside overlay, or by using SWIPE motion on mobile devices.
  • Lightbox Overlay can be closed by using X button inside overlay or by pressing Backspace or ESC key on your keyboard. On mobile devices, swiping up closes overlay.
  • When you reach end of image gallery and try to click next image, screen will vibrate to indicate that there is no more images available. Same applies to start of gallery with previous button.
  • Images inside LBO can be viewed in Fullscreen-mode by pressing Fullscreen -button. Image galleries can also be browsed in fullscreen-mode by using arrow keys on keyboard or by swiping on mobile devices.

Browser Support

Lightbox Overlay has been tested on 6 most popular browsers in 2024 and works correctly on all of them. Some older versions of these browsers might have outdated JavaScript and CSS decipher engine and Lightbox Overlay might not work as planned.

Supported Browsers:

  • Microsoft Edge
  • Mozilla Firefox (Web/Mobile)
  • Google Chrome (Web/Mobile)
  • Opera
  • Safari (Web/Mobile)
  • Samsung Internet (Mobile)
